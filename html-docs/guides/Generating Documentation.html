Generating Documentation

API documentation is generated using standard javadoc. Developer guides are generated using MultiMarkdown. Installation instructions are available here.

To build the documentation, open a Terminal in the root of the clone. Then run the following commands:

# Assuming you are in the root of BoseWearable-Android

$ ./gradlew generateCombinedJavadoc

This will generate the documentation under build/docs/javadoc/. Open the index.html file in a web browser to read the generated documentation.
