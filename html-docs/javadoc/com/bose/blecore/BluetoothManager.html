        public void startScanning(long timeout,
                          @NonNull
                          TimeUnit timeoutUnit,
                          @NonNull
                          ScanListener listener)
        Begins scanning for devices. The specified listener is invoked when a DiscoveredDevice is discovered or updated. Only one scan can be in progress at a time. The application must call stopScanning() in order to end an in-progress scan. Failure to do so causes this method to emit ScanError.ALREADY_STARTED error. The application should stop a scan after a device has been found.
        Parameters:
        timeout - The amount of time elapsed after an advertisement is received before considering the device lost and removing it
        timeoutUnit - The unit of timeout
        listener - The listener to notify about scan events

      • stopScanning

        public void stopScanning()
        Stops scanning for devices. This must be called before a subsequent scan is performed.
