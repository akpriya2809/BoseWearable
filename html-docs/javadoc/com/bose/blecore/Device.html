com.bose.blecore

Interface Device

  • All Known Subinterfaces:
    WearableDevice

    public interface Device
    A connected Device with an open Session to it. After a DiscoveredDevice has been found from scanning, use BluetoothManager.session(DiscoveredDevice) to create a Session to it. Once the Session has been successfully Session.open()-ed, an instance of Device is created and can be accessed from Session.device().
