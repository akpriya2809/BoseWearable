        public static final int REFRESH_REQUIRED
        Indicates inconsistent GATT services. Android caches GATT services and characteristics at pairing time. If remote device's properties change afterwards, these changes do not reach paired devices. In order to receive the updates, the remote device needs to be unpaired ("forgotten") and then paired again.
