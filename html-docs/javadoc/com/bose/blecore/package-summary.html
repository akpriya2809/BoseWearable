Package com.bose.blecore

Low level Bluetooth GATT abstractions

See: Description

  • Interface Summary 
    Interface Description
    Device
    A connected Device with an open Session to it.
    DeviceFactory  
    DeviceIdentification
    The DeviceIdentification interface provides the information necessary to identify a particular device type based upon its advertised services, its advertisement data, and the services a BluetoothDevice provides.
    ScanListener
    Listener for reporting Bluetooth scanning results and errors.
    ServiceIdentification
    Identifies a Service that is provided by a Device.
    Session
    A session represents the communication between this device and a remote BluetoothDevice.
    SessionBondingDelegate
    The bonding delegate gets notified when a device is required to be put into pairing mode in order to proceed with the connection.
    SessionDelegate
    A session delegate is notified of connectivity events related to a Session.
