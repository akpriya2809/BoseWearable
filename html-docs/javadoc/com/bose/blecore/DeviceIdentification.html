Interface DeviceIdentification


  • public interface DeviceIdentification

      • advertisementFilter

        boolean advertisementFilter(@Nullable
                            SparseArray<byte[]> advertisementData)

        Allows a device type to use advertisement data to augment the device IDENTIFICATION process. The BluetoothManager class scan for devices using the requiredAdvertisedServiceUuids() method to specify the advertised devices. When a BluetoothManager receives a scan result, it calls this function with the received manufacturer data to allow the device type the opportunity to say whether the discovered device is in fact the target type.

        In simple cases where there is a single device type that is adequately identified by the requiredAdvertisedServiceUuids() set, this function can immediately return true.

        Parameters:
        advertisementData - The advertisement data
        Returns:
        Returns true if this discovered device and advertisement represents a device of this type, false otherwise
