com.bose.blecore

Class SerializedGatt

  • java.lang.Object
    • com.bose.blecore.SerializedGatt

  • public class SerializedGatt
extends Object

    A wrapper class for BluetoothGatt that implements queueing of operations. Multiple operations (reads, writes, notification changes) can be queued and the operations run in serialized fashion, one after another.

    Additionally, implements a BluetoothGattCallback multiplexer, to register (and remove) multiple callbacks.

      • refreshServices

        public void refreshServices()
        Refresh GATT services. Android OS Bluetooth stack caches the list of available services and their characteristics for paired devices. New services or characteristics added (or removed) by the remote Bluetooth device firmware are thus not detected, requiring either (a) to unpair/pair cycle or (b) this method call.
