      • changeSensorConfiguration

        void changeSensorConfiguration(@NonNull
                               SensorConfiguration sensorConfiguration)
        Requests a change to the current sensor configuration. When this function is called, the specified sensor configuration value is written to the remote device. If the remote device accepts the new configuration, a WearableDeviceListener.onSensorConfigurationChanged(SensorConfiguration) event is fired. Then the sensorConfiguration() property is updated to the new value, which causes a WearableDeviceListener.onSensorConfigurationRead(SensorConfiguration) event to be fired. If the remote device rejects the new configuration, a WearableDeviceListener.onSensorConfigurationError(com.bose.wearable.BoseWearableException) event is fired.
        Parameters:
        sensorConfiguration - The new sensor configuration
